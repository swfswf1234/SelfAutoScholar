## 学习路径

1. **数学分析 + 线性代数** → **点集拓扑**（分析基础三件套）
2. **拓扑** → **实分析 + 复分析**（并行攻克 QE 重灾区）
3. **实分析** → **泛函分析**（谱理论核心链）
4. **数学分析** → **ODE → PDE**（方程方向）
5. 全部 → **Berkeley Problems**（QE 冲刺）
